# README_PC2 — Funcionamento Básico

## Objetivo
Validar o funcionamento individual das partes fundamentais do sistema utilizando o Raspberry Pi.

## Entregáveis

### Implementação
- Conectar os módulos principais ao Raspberry Pi  
- Testar cada componente individualmente  

Exemplos de módulos:
- Sensores (temperatura, presença, etc.)  
- GPS  
- Câmera  
- Comunicação (e-mail, rede, etc.)  

### Demonstração
- Comprovação de que cada módulo funciona corretamente  
- Testes individuais documentados  

### Tecnologias Permitidas
- Linguagens interpretadas (Python, Octave, etc.)  
- Bibliotecas prontas  

### Documento (Relatório Parcial)
O relatório deve conter:

- Descrição do hardware utilizado  
- Descrição do software desenvolvido  
- Resultados obtidos nos testes  

## Critérios de Avaliação
- Funcionamento correto dos módulos  
- Clareza na descrição técnica  
- Organização dos testes  
- Coerência entre hardware e software  

## Observações
- Não é necessário integração completa ainda  
- Foco na validação individual dos componentes
