# README_PC3 — Refinamento do Protótipo

## Objetivo
Integrar os módulos do sistema e implementar recursos básicos de sistemas operacionais.

## Entregáveis

### Implementação
- Integração das partes fundamentais do sistema  
- Funcionamento conjunto dos módulos  

### Recursos de Sistema
Implementar:
- Múltiplos processos e/ou threads  
- Pipes  
- Sinais  
- Semáforos  
- Mutex  

### Linguagem
- Desenvolvimento em C e/ou C++  

### Documento (Relatório Parcial)
O relatório deve conter:

- Atualização da descrição do hardware  
- Atualização da descrição do software  
- Resultados obtidos com o sistema integrado  

## Critérios de Avaliação
- Integração correta dos módulos  
- Uso adequado de recursos de sistema  
- Organização e clareza do código  
- Qualidade da documentação  

## Observações
- Início da transição para uma solução mais robusta  
- Código deve refletir conceitos vistos em sistemas operacionais
